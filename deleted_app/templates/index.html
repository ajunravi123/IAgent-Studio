Available Endpoints

Tools

  • GET /api/v1/tools - Get all tools
  • GET /api/v1/tools/{tool_id} - Get a specific tool
  • POST /api/v1/tools/add/{tool_id} - Add a tool
  • POST /api/v1/tools/custom - Create a custom tool

Agents

  • GET /api/v1/agents - Get all agents
  • GET /api/v1/agents/{agent_id} - Get a specific agent
  • POST /api/v1/agents - Create a new agent
  • PUT /api/v1/agents/{agent_id} - Update an agent
  • DELETE /api/v1/agents/{agent_id} - Delete an agent
